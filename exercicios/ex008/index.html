Principais Formatações

Negrito / Destaque

Nesta frase tenho um termo em negrito usando a tag <b> (Não semântica).

Nesta frase tenho um termo em destaque usando a tag <strong> (Semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (Não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (Semântica).

Texto Marcado

Podemos também fazer um texto marcado usando a tag <mark> (Semântica)

Podemos editar com CSS no próprio texto com style

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo, utilizando o <big> (Não semântico) e o <small> (Semântica).

Texto deletado

Podemos marcar um texto como excluído para saber que pode ser lido, mas não considerado, utilizando a tag <del> (Semântica)

Texto inserido / Sublinhado

Podemos marcar um texto como inserido para dar uma ênfase e indicar que foi adicionado depois utilizando a tag <ins> (Semântica).

Podemos sublinhar um texto utilizando a tag <u> (Não semântica).

Texto sobrescrito

Podemos sobrescrever para inserir coisas do tipo X20 + 3, utilizando a tag <sup> (Semântica).

Texto subscrito

Podemos subscrever para inserir coisas do tipo H2O, utilizando a tag <sub> (Semântica).
